# NUTRI TRADE LTD — Корпоративный B2B-лендинг

Адаптивный B2B-лендинг для болгарского поставщика орехов и семян. Проект разработан для оптовых клиентов, категорийных менеджеров торговых сетей и сегмента HoReCa.

---

## 🚀 Как посмотреть проект

Вы можете ознакомиться с лендингом двумя способами:

### Вариант 1: Быстрый просмотр (Онлайн)
Самый простой способ — открыть уже развернутую версию проекта в браузере:

👉 **https://arakelan-artur-first-project-softle.vercel.app/**

Сайт полностью адаптивен и корректно отображается как на компьютерах, так и на мобильных устройствах.

### Вариант 2: Локальный запуск (Через Visual Studio Code)
Если вы хотите посмотреть код или запустить проект локально:

1. **Скачайте проект** (нажмите зеленую кнопку *Code* -> *Download ZIP* и распакуйте архив).
2. **Откройте папку** проекта в редакторе [Visual Studio Code](https://code.visualstudio.com/).
3. **Используйте Live Server**:
   * В VS Code перейдите во вкладку расширений (`Ctrl+Shift+X`).
   * Найдите и установите расширение **Live Server** (автор: Ritwick Dey).
   * Откройте файл `index.html`.
   * Нажмите кнопку **"Go Live"** в правом нижнем углу или кликните правой кнопкой мыши по `index.html` -> *Open with Live Server*.
4. Сайт автоматически откроется в вашем браузере.
